# Project Brief: Interactive Hat Customizer

## 1. Strategy Core (North Star)
- **Client Name**: N/A (Internal/White Label Tool)
- **Industry**: Promotional Products / Swag
- **North Star (Singular Desired Outcome)**: Empower users to visualize and customize hat designs effortlessly, increasing engagement and order accuracy.
- **Offer (The "What")**: An interactive, step-by-step hat customization wizard.
- **ICP (The "Who")**: Businesses and individuals looking for custom branded headwear.
- **USP (The "Why Use")**: Visual, intuitive, and immediate feedback on customization options (patches, embroidery, colors).

## 2. Technical Scope
- **Source of Truth**: Local state/JSON configuration initially.
- **Integrations**: Potential future integration with e-commerce/order systems.
- **Delivery Payload**: React/Next.js Application (Interactive Form).

## 3. Behavioral Rules & Brand
- **Personality/Voice**: Professional, Creative, User-Friendly.
- **Behavioral Constraints**: 
    - Must guide user sequentially (Style -> Bill -> Panels -> Decoration).
    - Provide clear visual feedback for selected options.
