# Content & Copy

This directory acts as the central repository for all written content and raw media.

- **website-copy/**: Drafts and final copy for website pages (Home, About, etc).
- **funnel-copy/**: Scripts for VSLs, sales letters, and email sequences.
- **blog/**: Markdown files or drafts for blog posts.
- **media/**: Raw photos, video shoots, and unoptimized assets.
